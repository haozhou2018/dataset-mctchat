# dataset-mctchat
LLM-generated medical dialogue dataset sample for Medical Chinese Test

The excel file records a sample (1000 examples) of dataset MCT-Chat.

Complete MCT-Chat consists of 20358 medical dialogue examples, which were generated through LLMs and serve as potential testing materials for Medical Chinese Test (MCT).

Dataset Description:
(1) The 1st column is the used machine insturctions.
(2) The 2nd column is the corresponding generated dialogues.
(3) The 3rd column is the iteration number in using the generation framework.
